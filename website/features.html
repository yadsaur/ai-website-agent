Grounded trust

Stay honest when the site supports the answer, and equally honest when it does not

5minBot is built to stay within the boundaries of the processed site. When a real page supports the answer, it can show a clean source chip. When it cannot support an answer, it falls back instead of making things up.

  • Source chips only when a real supporting page exists.
  • No-answer mode for unsupported questions.
  • Cleaner markdown formatting for longer replies.
